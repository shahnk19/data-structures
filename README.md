# Data Structures

A collection of data-structures implementations in C++.

## Build

- [Install CMake](https://cmake.org/install/)
- Run following in terminal

### Build the project

```
  $ ./bin/build.sh
```

### Test the Linked List

```
  $ ./LinkedListTest
  $ ./LinkedListInsertionTest
  $ ./LinkedListDeletionTest
```

### Test the Stack

```
  $ ./StackTest
```
